<h2>Atualizacao - Emuladores (PT)</h2>
<p>Foram aplicadas melhorias para tornar o launcher mais estavel em alojamento estatico (Neocities).</p>
<ul>
	<li>Os links de lancamento passaram a usar hash (<code>#core=...</code>) em vez de query string (<code>?core=...</code>).</li>
	<li>O launcher universal passou a ler parametros apenas via hash para evitar versoes antigas em cache no servidor.</li>
	<li>A biblioteca de ROMs usa <code>emulators/universal/roms/catalog.json</code> para listar jogos de forma fiavel.</li>
	<li>Foi adicionado o botao <strong>Copy Launch Link</strong> para copiar o URL canonico em hash.</li>
</ul>
<p>Exemplo de URL recomendado: <code>emulators/universal/index.html#core=nes&amp;name=NES&amp;romDir=nes</code></p>
